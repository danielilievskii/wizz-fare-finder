# Wizz Fare Finder  

Wizz Fare Finder is a lightweight, budget-friendly alternative to the Wizz Air's official Fare Finder for quick round-trip fare searches from Skopje.

## Features  

- Automated flight scraping that keeps fares from Skopje up to date
- Smart filtering for instantly narrowing down trips by duration and budget
- Round-trip matching that pairs outbound and return flights automatically
- Nearby airport return matching to suggest round-trips with returns from airports close to your destination

## Tech Stack

- React.js (Typescript) - Interactive and responsive frontend interface
- Tailwind CSS - Utility-first styling
- ShadCN - UI components for better user experience
- Fast API - Backend that handles scraping, flight storage, and search logic
